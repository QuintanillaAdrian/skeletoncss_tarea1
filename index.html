Todo lo que hacemos es de código abierto

Nuestra plataforma consiste totalmente de componentes de código abierto y se basa en una plataforma sólida de programas libres (como GNU/Linux). Además, colaboramos de manera activa con el ecosistema para mejorarlo para todos.

Seguro y respetuoso de la privacidad

Como el código fuente está disponible para su revisión, cualquiera (un investigador de seguridad, un usuario preocupado o un fabricante que incluye el sistema en sus equipos) puede verificar que el software es seguro y no filtra información personal.

Hecho para los desarrolladores

Tanto si su aplicación puede beneficiarse de una función o API nueva del sistema, como si tiene curiosidad por saber cómo se ha creado una prestación o una pauta de diseño existentes, usted tiene acceso completo a nuestro código fuente. Cópielo, aprenda de él, modifíquelo y redistribúyalo.
